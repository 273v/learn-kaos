Let's run KAOS. This example pulls **typed, normalized legal citations** out of
plain text using [`kaos-citations`](https://github.com/273v/kaos-citations) — a
pure-Python package with a Rust-backed matcher. It's fully **deterministic** and
**offline**: no API key, no network model, same output every time.

## Run it

One command — no clone, no virtualenv:

```bash
uv run https://raw.githubusercontent.com/273v/learn-kaos/main/examples/citations-extract.py
```

Or from a local clone of this repo:

```bash
uv run examples/citations-extract.py
```

You'll see:

```text
Found 1 citation(s):

  kind=case     normalized='347 U.S. 483 (1954)'  span=(61, 73)
```

## What just happened

- The `# /// script` block at the top is [PEP 723](https://peps.python.org/pep-0723/)
  inline metadata. `uv` reads it, resolves `kaos-citations` into a cached isolated
  environment, and runs the file — that's why one command is enough.
- `extract_citations(text)` returns a list of typed `Citation` objects. Each one
  carries a `kind` (here `case`), the `raw` matched text, a `normalized` form, and
  an absolute `(start, end)` `span` into the input.
- The `assert`s at the bottom make the file **its own test**. This site runs it in
  CI, so this page is guaranteed to be correct.
