**Goal:** build a graph and find its most central nodes. `kaos-graph` is a Rust
(petgraph) engine with 40+ algorithms; here we use PageRank. Deterministic, offline.

```bash
uv run examples/graph-pagerank.py
```

**Notes**

- `pagerank(graph)` returns `NodeRank(node_id, score)` entries, highest first.
- Other algorithms live alongside it: `degree_centrality`, `betweenness_centrality`,
  `shortest_paths`, `topological_sort`, connectivity, cycles, and more.
- KAOS agents build a per-session knowledge graph (PROV-O + citations) you can query
  with these same algorithms plus SPARQL — see
  [query the session graph](/how-to/query-the-session-graph).
- The [graph-algorithms reference](/reference/graph-algorithms) lists the full catalog.
